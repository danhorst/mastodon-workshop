# mastodon-workshop
Ruby At Scale Workshop with Mastodon

## Usage

Check out this repo next to a mastodon repo, like so:

```
tree . -L 1
.
├── mastodon
└── mastodon-workshop
```

Run in the mastodon directory:

```
../mastodon-workshop/00_mastodon_test.sh && ../mastodon-workshop/01_create_admin_package.sh 
```
